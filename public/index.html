FLARE-ON 10 - Challenge 13

Posted on 2023-12-03 in unpacking, deobfuscating, malware analysis • 866 words • 5 minute read
Tags: malware, unpacking, deobfuscating, reverse engineering

The original post This challenge is a bind shell that opens port 1337. If we use netcat to connect to this port and wireshark to capture packets, we see that it receives windows commands from the client side and returns results. These data are all in unencrypted cleartext form. This is an obfuscated file so opening it in IDA will not properly parse the functions of this bind shell. To know all the functions of the bind shell, we first need to write a script to convert the obfuscated file into a form that IDA can read.
